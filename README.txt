# Yafet Wolde Portfolio

This repository contains the source code for Yafet Wolde's personal portfolio website. The portfolio showcases Yafet's skills, projects, and contact information.

## Website Overview

The portfolio website includes the following sections:
- **Intro**: A brief introduction about Yafet Wolde and a link to view recent projects.
- **Header**: The logo and navigation links.
- **Nav**: Links to different sections of the site and social media profiles.
- **Main**: Detailed information about featured projects.
- **Posts**: Additional projects with descriptions and links.
- **Contact**: Contact information including address, phone, email, and social media links.
- **Footer**: Copyright information and design credits.

## Technologies Used

- **HTML5**: Structure of the webpage.
- **CSS3**: Styling and layout of the webpage.
- **FontAwesome**: Icons used throughout the website.
- **Google Fonts**: Custom fonts for better typography.

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/japhettcw/YafetWoldePortfolio.git
    ```
2. Navigate to the project directory:
    ```sh
    cd YafetWoldePortfolio
    ```
3. Open `index.html` in your preferred web browser to view the portfolio.

## Project Structure

- `index.html`: The main HTML file for the portfolio.
- `assets/css/main.css`: The main CSS file for styling the portfolio.
- `assets/css/noscript.css`: CSS for when JavaScript is disabled.
- `images/`: Directory containing images used in the portfolio.
